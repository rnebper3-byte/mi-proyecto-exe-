Instrumentos de evaluación

En esta Situación de Aprendizaje dirigida, la evaluación se plantea como un proceso continuo, formativo y orientador, que permite recoger información sobre el aprendizaje del alumnado y ajustar la intervención educativa.

Para ello, se utilizarán diferentes instrumentos de evaluación que permiten valorar tanto el proceso como el producto final, así como la reflexión del alumnado sobre su propio aprendizaje.

Vamos a utilizar dos instrumentos de evaluación:

1. LISTA DE COTEJO (evaluación del proceso)

La lista de cotejo se utilizará el profesorado para evaluar el trabajo del alumnado durante el desarrollo del proyecto. La podemos descargar en el siguiente enlace:

https://drive.google.com/drive/u/0/folders/1du3jSB-sqCJxU9t_DmnBTafr5o2H_5Ah

2. CUESTIONARIO DE REFLEXIÓN DEL ALUMNADO

Este instrumento permite recoger información sobre:

●      Lo que ha aprendido el alumnado

●      Su percepción del proceso

●      Dificultades encontradas

●      Uso de herramientas digitales

Se realizará una vez finalizado el proyecto, mediante el siguiente enlace al cuestionario de Google Forms.

https://docs.google.com/forms/d/1FV3VzxUx1LN8rne1d11ofDZ76_qOWHk5ZFeekOOILnA/edit
